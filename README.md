# ~/startpage-command-center

This is a custom made start-page. It's minimal, terminal-inspired and still not there yet cause some QoL features are limited ( Mainly the gallery ). I decided to share it to the public.

##  Features


* **Smart Search**: Detects URLs and TLDs (like `.com`, `.org`) to navigate directly, otherwise defaults to Google.

* **Shorthand Commands**:
    * `y! [query]` — Search YouTube directly.
    
* **Dynamic Artwork**: A vertical gallery with a custom-built upward-opening menu. (limited/handful of images)

* **Auto-Cycle Mode**: Optional slideshow mode that rotates backgrounds every 30 seconds. (limited)

* **Adaptive Greeting**: Changes based on the time of day. 
  Time Range	Greeting
  05:00 – 11:59	"Good morning."
  12:00 – 17:59	"Good afternoon."
  18:00 – 21:59	"Good evening."
  22:00 – 04:59	"Late night, huh?"
  
* **Keyboard Driven**:
    * **Any key**: Immediately focuses the search bar.
    * **Tab**: Cycles to the next artwork.
    * **Esc**: Clears the search bar and removes focus.


##  Setup

* 'JetBrains Mono' typeface and a 'Dracula-inspired' color palette. (Linux inspired)
